# core — the OS layer

`core/` holds the NixOS modules, library, and smoke-test host that make
up the glixos OS layer. These outputs are exposed by the **repo-root
flake** at `../flake.nix`; `core/` itself is not a standalone flake.

## What's here

```
core/
├── lib/
│   └── importManifest.nix    # manifest TOML → module lists
├── modules/
│   ├── core/                 # base system: boot, nix, networking, users, locale, glix CLI
│   │   ├── default.nix       # aggregator (function of `inputs`)
│   │   ├── boot.nix
│   │   ├── locale.nix
│   │   ├── networking.nix
│   │   ├── nix.nix           # flakes on, channels off
│   │   └── users.nix
│   └── desktop/
│       └── default.nix       # optional desktop profile
└── hosts/
    └── vm/                   # internal QEMU smoketest host
        └── default.nix
```

## Public surface (consumed via the root flake)

| Output                            | What you get                                                       |
|-----------------------------------|--------------------------------------------------------------------|
| `lib.importManifest`              | Pure function: `(manifestPath, inputs, defaultUser) → { systemModules, homeModules, homeModulesByUser, manifest }`. |
| `nixosModules.glixos`             | Base OS module; import from every host. Installs `pkgs.glix` automatically. |
| `nixosModules.desktop`            | Optional desktop profile.                                          |
| `nixosModules.default`            | Alias for `nixosModules.glixos`.                                   |
| `nixosConfigurations.vm`          | Internal smoketest host. Not the user-facing pattern.              |
| `packages.${system}.vm`           | A runnable QEMU image built from the smoketest host.               |

## Building the smoketest

From the repo root:

```bash
nix flake check          # builds the VM toplevel + the glix derivation
nix build .#vm           # produces ./result/bin/run-*-vm
./result/bin/run-*-vm    # boots it
```

## Using the OS layer from a user-packages flake

Generated by `glix init`; you don't normally write this by hand:

```nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
    glixos-core = {
      url = "github:powerreddude/glixos";
      inputs.nixpkgs.follows = "nixpkgs";
    };
    home-manager.url = "github:nix-community/home-manager";
  };

  outputs = { self, nixpkgs, glixos-core, home-manager, ... }@inputs: {
    nixosConfigurations.laptop = nixpkgs.lib.nixosSystem {
      system = "x86_64-linux";
      specialArgs = { inherit inputs; };
      modules = [
        glixos-core.nixosModules.glixos
        home-manager.nixosModules.home-manager
        ./hosts/laptop
      ] ++ (glixos-core.lib.importManifest {
        manifestPath = ./hosts/laptop/glix.toml;
        inherit inputs;
        defaultUser = "alice";
      }).systemModules;
    };
  };
}
```
